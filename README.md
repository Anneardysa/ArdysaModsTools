## üì• Installation

### Requirements

-  Windows 10/11 (64-bit)
-  .NET 8.0 Desktop Runtime
-  Dota 2 installed via Steam

### Download

1. Get the latest installer from [Releases](https://github.com/Anneardysa/ArdysaModsTools/releases)
2. Run `ArdysaModsTools_Setup_x64.exe`
3. Follow the installation wizard

> The installer will automatically check for .NET 8.0 and WebView2 Runtime.

## üöÄ Quick Start

1. **Close Dota 2** if running
2. **Launch AMT 2.0** (run as Administrator)
3. **Click "Install ModsPack"** to apply all mods
4. **Start Dota 2** and enjoy!

### After Dota 2 Updates

Run AMT and click **"Install ModsPack"** again to re-apply your mods.

### Disable Mods

Click **"Disable Mods"** to remove all modifications and restore vanilla Dota 2.

## ‚ùì FAQ

**Q: Is this safe to use?**

> Yes, AMT only modifies local cosmetic files. It doesn't interact with online services.

**Q: Will I get banned?**

> Cosmetic mods generally don't result in bans, but policies can change. Use at your own risk.

**Q: Can other players see my mods?**

> No, mods are client-side only. Others see default game assets.

**Q: Mods not showing after Dota 2 update?**

> Run AMT and click "Install ModsPack" to re-apply.
