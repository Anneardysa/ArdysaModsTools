# User Quick Reference

Quick reference guide for AMT 2.0 users.

> **📖 For complete documentation, see the [Main README](../../README.md)**

---

## ⚡ Quick Start

1. **Close Dota 2**
2. **Launch AMT 2.0**
3. **Click "Auto Detect"**
4. **Install mods** - Choose Skin Selector or Miscellaneous
5. **Click "Patch Update"**
6. **Launch Dota 2** and enjoy!

---

## 🎯 Main Features

### Skin Selector

Install custom hero cosmetic sets from curated library.

### Miscellaneous Mods

Apply weather, terrain, HUD, music, and cursor mods.

### Auto-Patching

Keep mods working after Dota 2 updates with one click.

### Auto-Detection

Automatically finds your Dota 2 installation path.

---

## 🔧 Common Tasks

### Install Hero Skins

```
Skin Selector → Select Hero → Choose Set → Install → Patch
```

### Apply Weather/Terrain

```
Miscellaneous → Select Category → Choose Mod → Install → Patch
```

### Update After Dota Patch

```
Patch Update → Confirm
```

### Disable All Mods

```
Disable Mods → Restart Dota 2
```

---

## ⚠️ Important Notes

-  **Always close Dota 2** before using AMT
-  **Run as Administrator** if you encounter permission errors
-  **Patch after updates** - Run "Patch Update" after Dota 2 patches
-  **Use at own risk** - Modding may violate Valve's ToS

---

## 🆘 Troubleshooting

| Issue               | Solution                        |
| ------------------- | ------------------------------- |
| Can't detect Dota 2 | Use "Browse" to select manually |
| Mods not showing    | Click "Patch Update"            |
| Permission errors   | Run as Administrator            |
| After Dota update   | Click "Patch Update"            |
