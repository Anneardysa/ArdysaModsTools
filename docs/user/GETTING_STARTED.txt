  FIRST TIME SETUP (Takes 5 minutes)
â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•

STEP 1: LAUNCH THE APPLICATION
  - Close Dota 2 if it's running
  - Launch ArdysaModsTools from Start Menu or Desktop

STEP 2: DETECT DOTA 2 INSTALLATION
  - Click "Auto Detect" button
  - The app will automatically find your Dota 2 installation
  - If detection fails, click "Manual Select" and browse to:
    C:\Program Files (x86)\Steam\steamapps\common\dota 2 beta

STEP 3: INSTALL MODS
  - Click "Install" button
  - Choose "Auto Install" (recommended)
  - Wait for download and installation to complete
  - Status will show "Ready" in green when finished

STEP 4: LAUNCH DOTA 2
  - Close ArdysaModsTools
  - Launch Dota 2 normally from Steam
  - Your mods are now active!

1. MOD INSTALLATION
   
   Install curated mod packs with cosmetic enhancements.
   
   How to use:
   - Click "Install" button
   - Choose "Auto Install"
   - Wait for completion
   
   To disable:
   - Click "Disable" button
   
   To update after game patches:
   - Click "Patch Update"
   - Select "Quick Patch"

2. HERO SET GENERATION
   
   Create custom hero skins from community sets.
   
   How to use:
   - Click "Select Hero" button
   - Browse or search for your hero
   - Select a set from the dropdown
   - Click "Generate"
   - Wait 2-5 minutes per hero
   
   Tips:
   - Star your favorite heroes for quick access
   - Generate multiple heroes at once

3. MISCELLANEOUS MODS
   
   Customize weather, terrain, HUD, and audio.
   
   How to use:
   - Click "Miscellaneous" button
   - Choose generation mode:
     * Clean Generate: Start fresh
     * Add to Current: Add to existing mods
   - Select your options
   - Click "Generate"

PROBLEM: Can't launch ArdysaModsTools
SOLUTION: Make sure Dota 2 is completely closed. Check Task Manager for
          "dota2.exe" and end the process if found.

PROBLEM: Auto detection fails
SOLUTION: Use "Manual Select" and browse to your Dota 2 folder.
          Default: C:\Program Files (x86)\Steam\steamapps\common\dota 2 beta

PROBLEM: Mods not visible in Dota 2
SOLUTION: 
  1. Check status shows "Ready" (green)
  2. Click "Patch Update" â†’ "Full Patch"
  3. Restart Dota 2 completely

PROBLEM: After Dota 2 update, mods stopped working
SOLUTION:
  1. Launch ArdysaModsTools
  2. Click "Patch Update"
  3. Select "Quick Patch"
  4. Wait for completion
  5. Relaunch Dota 2

PROBLEM: Permission errors
SOLUTION: Right-click ArdysaModsTools and "Run as Administrator"

PROBLEM: .NET 8 not installed
SOLUTION: The installer will prompt you to download it.
          Get it from: https://dotnet.microsoft.com/download/dotnet/8.0
          Install "Desktop Runtime x64" version

Q: Is this safe to use?
A: It only modifies cosmetic files, but use at your own risk. We cannot
   guarantee immunity from game restrictions.

Q: Can I get VAC banned?
A: Cosmetic mods typically don't trigger VAC, but we cannot provide guarantees.

Q: Where are mods installed?
A: In your Dota 2 folder: game/dota/_ArdysaMods/pak01_dir.vpk

Q: How do I uninstall?
A: Click "Disable" in ArdysaModsTools, then uninstall via Windows Settings.

Q: Can I use multiple hero sets at once?
A: Yes! That's the main feature. Select different sets for different heroes.

Q: How do I go back to vanilla Dota 2?
A: Click "Disable" button, or verify game files in Steam.

Q: Does this work on Mac or Linux?
A: No, Windows-only (Windows 10/11 64-bit).
